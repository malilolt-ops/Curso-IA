# sensibilidad-demanda

Aplicación en Streamlit para calcular `Demanda_base` y simular sensibilidad al precio a partir de un archivo Excel.

## Ejecutar

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Repositorio de referencia

Este módulo está preparado para usarse en: https://github.com/malilolt-ops/Modelo-Demanda
